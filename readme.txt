Plugin for displaying schema FAQ asked questions in WordPress .

== Description ==

This plugin is for displaying schema FAQ on the website  
1- Install the plugin
2- A metabox is created
3- Create and save questions and answers
4- The question and answer schema is automatically displayed to Google
5- Change location and title with short code [MS-FAQ title='']  
 
== Frequently Asked Questions ==

= What to do after installation? =

1- > wp-admin, go to the sections and find Metabox add MAgic Schema
2-Create FAQ and publish
